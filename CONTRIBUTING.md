Contributing

Thank you for deciding on contributing to this project!

📜 Guidelines

    Keep it Vanilla: Try to avoid adding too many external dependencies. We want the tool to be easy for anyone to run.

    OOP Design: We are using Object-Oriented Programming to keep the code clean. Please follow the existing class structures (RomManager, TileManager, etc.).

    Document Your Work: If you add a new feature, please add a quick comment explaining how the math works.


Getting Started

    Fork the repository.

    Create a Feature Branch.

    Submit a Pull Request with a description of what you changed.
    
The Roadmap:

The roadmap is small, and anything that is added to the [Python Version](https://github.com/AIMYII/Mega-Man-Acid-Tile-Investigator-Python/tree/master) of the project will be added here.

For now. I would request that a graphical user interface is made, and a tile manager(e.g., Dear ImGui or Raylib)

Port Logic: Take the working Python functions (like HexToDec or the TileManager math) and implement them in C++.
